11. Container With Most Water

Description

https://leetcode.com/problems/container-with-most-water/description/

Category

Array, Two pointers

Explain

Start with widest range, in order to contain more water (larger area), the next bound should be higher. In order to achieve this, the lower bound must be updated since it actually decide the current height of container. Also, since it is from sides to middle, current max is the max for this lower bound, and this bound is no longer userful.

My Solution

Time complexity

O(N)
