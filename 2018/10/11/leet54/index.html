54. Spiral Matrix

Description

https://leetcode.com/problems/spiral-matrix/description/

Category

Array

Explain

  1. Split the spiral into 4 part, each part could be [n-1, n-1, n-1, n-1] or [n, n-2, n, n-2]
  2. Special condition, if number of layers, min(x, y), is odd, the last loop should be done 1 round instead of 4.
