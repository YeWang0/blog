Detect blank snapshot

Backgroud:

A new model for blank page detection.

Solution:

Based on the html, IR based detection is built. However, the recall is around 60% in order to maintain a high precision(>90%)

During the process of building IR detection model, we find that the ‘no result’ remind provided by different websites could be a strong signal. As a result, we create a config file to record the ‘no result’ string on different sites and do detection by matching the string. With this rules based model, the precision is improved to 95% and recall is 85%.
