Update database automatically daily

Backgroud:

The blank page detection system need to use the demo blank page to compare the difference between a new search result and demo blank page. It is used as a signal for detection. However, the demo blank page could change as time goes by, so how to keep the demo blank page data updated automatically. <!-- more -->

Possible soultion:

  1. Database: mongoDB or Hbase. How to keep historical data?
  2. Trigger of execution: Time.sleep or Task scheduler
